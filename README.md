# Getting Started

bower + bundler + npm一括インストール

     $ sh ./setup.sh


# server (watch)

    $ gulp server
    
もしくは
    
    $ npm run dev #cleanなし gulp server --api develop --noclean trueのショートハンド
    $ npm run clean_dev #cleanあり
    
# build

    $ gulp build
    
# deploy

    $ npm run deploy
